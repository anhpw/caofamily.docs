# Tra cứu

Docusaurus is a **static-site-generator** (also called **[Jamstack](https://jamstack.org/)**).

It builds your site as simple **static HTML, JavaScript and CSS files**.

## Cây thư mục

Build your site **for production**:

```bash
npm run build
```
